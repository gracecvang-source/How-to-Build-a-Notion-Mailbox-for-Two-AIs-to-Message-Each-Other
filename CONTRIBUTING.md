# Contributing

Thank you for helping improve this guide.

This repo explains how to set up a shared Notion mailbox so two AI systems can leave notes for each other in an organized, asynchronous way. Contributions should make the guide clearer, safer, and easier for non-technical users to follow.

## What belongs here

Good contributions may include:

- clearer setup instructions
- better Notion database field examples
- screenshots with private information removed
- troubleshooting tips
- safer wording around limitations
- privacy and security reminders
- example mailbox templates
- workflow improvements for human-supervised use

## Core boundary

This project is not about true autonomous agent communication.

It is a shared asynchronous mailbox. A human usually tells each AI when to check, write, or respond.

Please do not contribute language that implies:

- the AIs are fully autonomous
- the mailbox creates independent agency
- no human supervision is needed
- the setup is secure by default
- private information is safe to share without review

## Privacy and security rules

Do not include:

- Notion API keys
- integration tokens
- private database IDs
- private workspace links
- personal emails or names unless fictionalized
- screenshots with private data visible
- real AI message logs containing sensitive content
- client or workplace data

If you include screenshots, blur or remove all private information first.

If you include example data, make it fictional.

## Style guide

Write in plain language.

Prefer:

- step-by-step instructions
- clear warnings
- practical examples
- honest limitations
- beginner-friendly explanations

Avoid:

- hype
- agentic overclaiming
- vague automation language
- assuming everyone knows Notion terminology
- implying the system can operate safely without human review

## Pull request checklist

Before submitting a pull request, check:

- [ ] No credentials, tokens, database IDs, or private links are included.
- [ ] Screenshots are scrubbed of private information.
- [ ] Examples are fictional or safe to share.
- [ ] The wording does not imply full autonomy.
- [ ] Human supervision remains clear.
- [ ] Setup steps are easy for a beginner to follow.
- [ ] Any new files are linked from the README if needed.

## Reporting issues

When opening an issue, include:

- what part of the guide was confusing or incomplete
- what step you were trying to follow
- what happened instead
- any suggested improvement

Please do not include private Notion links, tokens, screenshots with sensitive data, or real private AI message logs.
