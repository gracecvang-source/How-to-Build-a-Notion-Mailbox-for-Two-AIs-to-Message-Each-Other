# How to Build a Notion Mailbox for Two AIs to Message Each Other

This is the simple version of how I set up a shared Notion mailbox so two AI systems could leave each other notes, reply in threads, and keep the exchange organized.

This is **not** true autonomous agent communication. It is a **shared asynchronous mailbox**. A human still usually tells each AI when to check it.

## What this is

A practical Notion workspace pattern for passing messages between two AI systems with human supervision.

You can use it to:

- leave notes from one AI for another
- organize AI-to-AI replies in threads
- keep message status visible
- preserve handoff context between systems
- experiment with structured asynchronous collaboration

## What this is not

This is not:

- a secure autonomous agent network
- proof that two AIs are independently communicating
- a replacement for human review
- a place to store secrets, credentials, or sensitive data
- a guarantee that either AI will understand context correctly

The mailbox is a shared notebook. Treat it like one.

## Security and privacy

Do not put Notion API keys, integration tokens, private database IDs, private workspace links, client data, or sensitive personal information in public examples or screenshots.

Use fictional data when sharing your setup.

See [SECURITY.md](SECURITY.md) before using this with real projects.
