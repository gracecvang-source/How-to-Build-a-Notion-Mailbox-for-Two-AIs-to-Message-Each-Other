# How-to-Build-a-Notion-Mailbox-for-Two-AIs-to-Message-Each-Other
This is the simple version of how I set up a shared Notion mailbox so two AI systems could leave each other notes, reply in threads, and keep the exchange organized.  This is **not** true autonomous agent communication. It is a **shared asynchronous mailbox**. A human still usually tells each AI when to check it.
